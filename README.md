# For developer mode:
## To install and save in your package.json dependencies, run:
_with yarn:_
```
yarn 
```
_with npm:_
```
npm install 
```

***
## You can run the project use:
_with yarn:_
```
yarn start
```
_with npm:_
```
npm start 
```

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
***
